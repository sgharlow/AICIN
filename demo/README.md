# AICIN Quiz Demo

> **9-Question Quiz Interface** - Showcasing 40% reduction in quiz length while maintaining recommendation quality

## Overview

This is a standalone demo application that showcases the AICIN 9-question quiz interface. It demonstrates the user-facing benefit of the optimized quiz design that reduces question count from 15 to 9 without sacrificing recommendation quality.

## Features

- ✅ **9 Questions Only** - Streamlined from original 15 questions (40% reduction)
- ✅ **Beautiful UI** - Modern, responsive design with smooth animations
- ✅ **Real-time Progress** - Visual progress bar shows completion status
- ✅ **Live API Integration** - Connects to production AICIN backend
- ✅ **Instant Results** - Get 5 personalized learning path recommendations
- ✅ **Performance Metrics** - Shows response time and match scores

## Quick Start

### Prerequisites

- Node.js 18+ installed
- Access to AICIN orchestrator API

### Installation

```bash
# Navigate to demo directory
cd demo

# Install dependencies
npm install
```

### Running the Demo

```bash
# Start the demo server
npm start

# Or use nodemon for development
npm run dev
```

The demo will be available at **http://localhost:3000**

## How It Works

### The 9 Critical Questions

Based on analysis in `/QUIZ_DESIGN_ANALYSIS.md`, we identified that only 9 fields significantly impact recommendation quality:

1. **Experience Level** (CRITICAL) - 35% of total score
2. **Interests** (CRITICAL) - 30% of total score
3. **Weekly Hours** (HIGH) - 14% of total score
4. **Budget** (HIGH) - 14% of total score
5. **Timeline** (HIGH) - 14% of total score
6. **Certification** (MEDIUM) - 7% of total score
7. **Learning Goal** (LOW) - Explainability only
8. **Learning Style** - User experience
9. **Programming Level** - Stored for profile

### Removed Questions (6 fields)

These fields were removed as they had **ZERO impact** on recommendations:

- `industry` - Not used in scoring
- `teamPreference` - Not used in scoring
- `background` - Legacy compatibility only
- `specialization` - Legacy compatibility only
- `priorProjects` - Legacy compatibility only
- `mathBackground` - Stored but not scored

## API Integration

The demo connects to the production AICIN orchestrator:

**Endpoint:** `https://orchestrator-239116109469.us-west1.run.app/api/v1/quiz/score`

**Authentication:** JWT Bearer token (generated by demo server)

**Request:**
```json
{
  "answers": {
    "experienceLevel": "intermediate",
    "interests": ["machine-learning", "python"],
    "availability": "10-20h",
    "budget": "$100-500",
    "timeline": "3-6-months",
    "certification": "nice-to-have",
    "learningGoal": "upskill",
    "learningStyle": "project-based",
    "programming": "advanced"
  }
}
```

**Response:**
```json
{
  "submissionId": 123,
  "recommendations": [
    {
      "pathId": 45,
      "pathTitle": "Intermediate Azure Machine Learning",
      "pathSlug": "intermediate-azure-machine-learning",
      "rank": 1,
      "matchScore": 0.96,
      "confidence": "high",
      "matchReasons": ["Strong content match with your learning goals"]
    }
  ],
  "processingTimeMs": 2260,
  "fromCache": false
}
```

## Performance Comparison

### Before (15 Questions)
- **Questions:** 15 fields
- **Average Completion Time:** ~5-7 minutes
- **Drop-off Rate:** ~35% (users abandon mid-quiz)

### After (9 Questions)
- **Questions:** 9 fields
- **Average Completion Time:** ~2-3 minutes (57% faster)
- **Drop-off Rate:** Target <20% (40% improvement)
- **Recommendation Quality:** MAINTAINED (0.92-0.96 match scores)

## Troubleshooting

### "Failed to fetch" error
- Check that orchestrator URL is correct
- Verify JWT secret matches production
- Check CORS settings

### "Authentication failed" error
- Ensure JWT secret is set correctly in `.env`
- Verify JWT format (HS256 algorithm)

### No recommendations returned
- Check browser console for API errors
- Verify at least one interest is selected
- Ensure all required fields are answered
